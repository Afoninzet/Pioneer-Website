# Пионеры Дальнего Космоса — Сайт группы

## Описание

Веб-сайт для музыкальной группы с функцией управления репетициями и публикации новостей. Музыканты могут отмечать свою доступность в календаре, а также создавать и удалять новости. Все изменения синхронизируются между устройствами через Firebase и localStorage.

## Функциональность

- **Тёмная космическая тема** — весь сайт выполнен в тёмных тонах с анимированными звёздами
- **Авторизация музыкантов** — 4 учётные записи (Dima, Piter, Kirill, Ilya, пароль совпадает с логином)
- **Управление репетициями** — календарь на текущий и следующий месяц с отметками участников
- **Управление новостями** — авторизованные музыканты могут:
  - Публиковать новости (заголовок + текст)
  - Удалять любые новости (при наведении на карточку появляется кнопка 🗑️)
  - Все новости синхронизируются между устройствами
- **Синхронизация** — двойное хранилище (localStorage + Firebase), данные сохраняются даже при отсутствии интернета
- **Новости с модальными окнами** — клик по новости открывает полный текст
- **Раздел "Фото"** — пока пуст, выводит сообщение "Фото пока нет"

## Новые возможности в версии 3.0

1. **Публикация новостей** — форма создания новостей находится в разделе "Управление"
2. **Удаление новостей** — любой авторизованный музыкант может удалить любую новость
3. **Синхронизация через Firebase** — новости автоматически синхронизируются между всеми устройствами
4. **Офлайн-режим** — при отсутствии Firebase новости сохраняются в localStorage
5. **Автор не указывается** — публикация анонимная, только заголовок, дата, время и текст

## Технологии

- HTML5
- CSS3 (тёмная тема, адаптивная вёрстка)
- Vanilla JavaScript
- Firebase Realtime Database
- localStorage (резервное хранилище)

## Запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в любом современном браузере
3. Для работы календаря, новостей и авторизации требуется поддержка localStorage
4. Для синхронизации между устройствами необходим доступ к Firebase (интернет)
