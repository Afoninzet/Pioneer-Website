# Пионеры Дальнего Космоса — Сайт группы

## Описание
Веб-сайт для музыкальной группы с функцией управления репетициями. Музыканты могут отмечать свою доступность в календаре, и все участники видят отметки друг друга (синхронизация через localStorage).

## Функциональность
- **Тёмная тема** — весь сайт выполнен в тёмных тонах
- **Авторизация музыкантов** — 4 учётные записи (Dima, Piter, Kirill, Ilya, пароль совпадает с логином)
- **Управление репетициями** — календарь на текущий и следующий месяц
- **Отметки участников** — клик по дню отмечает/снимает отметку текущего пользователя
- **Синхронизация** — все участники видят отметки друг друга через единое хранилище localStorage
- **Иконка замка** — кнопка входа в виде небольшой иконки вместо текста

## Учётные записи
| Логин   | Пароль  | Отображаемое имя |
|---------|---------|------------------|
| Dima    | Dima    | Дима             |
| Piter   | Piter   | Пётр             |
| Kirill  | Kirill  | Кирилл           |
| Ilya    | Ilya    | Илья             |

## Технологии
- HTML5
- CSS3 (тёмная тема, адаптивная вёрстка)
- Vanilla JavaScript
- localStorage для хранения сессий и отметок

## Запуск
1. Скачайте все файлы проекта
2. Откройте `index.html` в любом современном браузере
3. Для работы календаря и авторизации требуется поддержка localStorage

## Примечание
Данные о доступности участников синхронизируются между всеми пользователями, так как используют общий ключ в localStorage. Это означает, что на одном устройстве все музыканты будут видеть актуальную картину занятости.
